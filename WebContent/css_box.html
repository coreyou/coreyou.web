方框尺寸

height和width屬性：
指定方框的尺寸最常使用像素、百分比或em。
在過去，像素是最熱門的方法，因為可以精確控制方框大小。
當你使用百分比時，方框的尺寸和瀏覽器視窗是相關的。若方框是在另一個方框內，那麼它指的就是容器方框的百分比。
當你使用em時，方框尺寸是以容器的大小為基準的。最近設計師在因應各種不同尺寸螢幕的裝置進行設計時，有開始改採百分比和em來當作單位的現象。
下面的例子中，如果<p>位於<div class="box">裡面，那麼其75%就會等於500px的75%，也就是375px。

div.box {
 height: 500px;
 width: 500px;
 background-color: #bbbbaa;
}
p {
 height: 75%;
 width: 75%;
 background-color: #0088dd;
}

限制寬度

min-width: 當瀏覽器視窗很窄時，方框至多只能縮小到哪個尺寸。
max-width: 當瀏覽器視窗很寬時，方框至多只能放大到哪個尺寸。

td.description {
 min-width: 450px;
 max-width: 650px;
 text-align: left;
 padding: 5px;
 margin: 0px;
}

邊框寬度

border-width屬性: 可以使用像素為單位，也可以接受以下三種值，thin、medium、thick。
你也可以使用下列四個屬性來分別控制各邊框的尺寸: border-top-width、border-right-width、border-bottom-width、border-left-width。
也可以在一個屬性中一次為四個邊框指派不同的寬度: border-width: 2px 1px 1px 2px; (此方框的上邊框為2px、右邊框為1px、下邊框為1px、左邊框為2px)。
發現在firefox中單純指定邊框寬度，卻沒有指定邊框樣式，是不會出現邊框的。

p.one {
 border-width: 2px;
}
p.two {
 border-width: thick;
}
p.three {
 border-width: 1px 4px 12px 4px;
}

邊框樣式

border-style屬性: 可以接受下列值:
solid 單實線
dotted 一組方格點(如果你的邊框是2px寬，那麼點就是2px方格然後每個點之間有2px的間距)
dashed 一組短線
double 兩條實線(border-width屬性值是兩條線的總和)
groove 看起來像是刻入頁面
ridge 看起來像是凸出頁面
inset 看起來像是嵌入頁面
outset 看起來像是浮雕
hidden/none 無邊框
你也可以使用下列四個屬性來分別控制各邊框的樣式: border-top-style、border-left-style、border-right-style、border-bottom-style。

p.one {
 border-width: 5px;
 border-style: solid;
}
p.two {
 border-width: 5px;
 border-style: dotted;
}
p.three {
 border-width: 5px;
 border-style: dashed;
}
p.four {
 border-width: 5px;
 border-style: double;
}
p.five {
 border-width: 5px;
 border-style: groove;
}
p.six {
 border-width: 5px;
 border-style: ridge;
}
p.seven {
 border-width: 5px;
 border-style: inset;
}
p.eight {
 border-width: 5px;
 border-style: outset;
}

p.one {

內距

內容和邊框之間應該保持多少空間。
padding屬性: 通常以像素為單位，如果使用百分比為單位的話，表示的是瀏覽器視窗的百分比或是容器方框的百分比。
你也可以使用下列四個屬性來分別控制各邊框的內距: padding-top、padding-right、padding-bottom、padding-left。
你也可以使用速記來指定(以順時針為方向: 上、右、下、左): padding: 10px 5px 3px 1px;
請注意: 內距會直接增加方框的寬度，當方框已經有指定寬度的時候要注意。
padding屬性的值並不會像font-family屬性的color值一樣，被子元素繼承，所以你必須在每個需要內距的元素上指定padding。

p.one {
 padding: 10px 5px 3px 1px;
}

邊界

margin屬性: 方框之間的空隙。通常以像素為單位，也可以使用百分比或em。
你也可以使用下列四個屬性來分別控制各邊框的內距: margin-top、margin-right、margin-bottom、margin-left。
如果一個方框在另一個方框上面，邊界就會交疊，意思就是，兩者之間比較大的邊界就會被使用，較小的邊界就會被丟棄。
你也可以使用速記來指定(以順時針為方向: 上、右、下、左): margin: 1px 2px 3px 4px;
你也可以另外一種速記來指定(左右、上下): margin: 10px 20px;
請注意: 如果方框指派了一個寬度，邊界會加到方框寬度之上。
margin屬性的值並不會像font-family屬性的color值一樣，被子元素繼承，所以你必須在每個需要邊界的元素上指定margin。

p.one {
 margin: 20px;
}

將內容置中對齊

如果你想將一個方框放在頁面中央，你可以將left-margin和right-margin設定為auto。
為了將方框置中，你需要為方框設定一個寬度(否則他會佔據整個頁面的寬度)。
方框的寬度設定好之後，將左右邊界設為auto就會使瀏覽器在方框的兩側加入等寬的空隙。
為了讓這個功能在較舊的瀏覽器中正確顯示，容器方框必須有一個text-align: center;
text-align會繼承給子元素，因此如果你不希望方框中的文字也置中的話，就需要在方框中設定text-align屬性。

body {
 text-align: center;
}
p {
 width: 300px;
 border: 3px solid #0088dd;
 padding: 50px;
}
p.two {
 margin: 20px auto 20px auto;
 text-align: left;
}

更改行內/區塊狀態

display屬性。可以讓你將一個行內元素轉為區塊層級元素，或相反之。他也可以用來隱藏頁面中的元素。
display: inline;(使區塊層級元素表現出行內元素的行為)
display: block;(使行內元素表現出區塊元素的行為)
display: inline-block;(這會使區塊層級元素和行內元素一樣流動，但是依然保持區塊層級元素的其他特性)
display: none;(隱藏元素，但是在原始碼檢視的時候還是看的到方框內容)
如果你使用display這個屬性的話，行內方框並不會製作出區塊層級元素。

<html>
 <head>
  <meta charset="UTF-8">
  <title>Display</title>
  <style type="text/css">
   li {
    display: inline;
    margin-right: 10px;
   }
   li.coming_soon {
    display: none;
   }
  </style>
 </head>
 <body>
  <ul>
   <li>Home</li>
   <li>Products</li>
   <li class="coming_soon">Services</li>
   <li>About</li>
   <li>Contact</li>
  </ul>
 </body>
</html>

隱藏方框

visibility屬性。可以讓你將方框隱藏不見，但將原有位置空下來。
visibility: hidden;(隱藏元素)
visibility: visible;(顯示元素)
如果元素的visibility是hidden，原來應該呈現的位置就會是空白，如果你不希望出現空白，則應該使用display: none;

<html>
 <head>
  <meta charset="UTF-8">
  <title>Display</title>
  <style type="text/css">
   li {
    display: inline;
    margin-right: 10px;
   }
   li.coming_soon {
    visibility: hidden;
   }
  </style>
 </head>
 <body>
  <ul>
   <li>Home</li>
   <li>Products</li>
   <li class="coming_soon">Services</li>
   <li>About</li>
   <li>Contact</li>
  </ul>
 </body>
</html>

CSS3: 邊框影像

border-image屬性。他會將一張背景切成九片。
border-image屬性是下列屬性的速記: border-image-source、border-image-slice、border-image-width、border-image-outset、border-image-repeat。
語法: border-image: url("images/border.jpg") 11 11 11 11 stretch||round||repeat;
屬性中的資訊第一個為影像的URL，第二個為切割的位置，第三個是stretch、repeatround、round三者擇一，
stretch會延伸影像；repeat會重複影像；round也會重複影像，但如果拼接不完全吻合，它就會縮放拼接影像使之吻合。
-moz-border-image和-webkit-border-image並不在CSS規格內，但可以幫助舊版的Chrome、Firefox和safari。

body {
 text-align: center;
}
p.one {
 border-style: solid;
 border-width: 20px;
 -moz-border-image: url("images/border.jpg") 11 11 11 11 stretch;
 -webkit-border-image: url("images/border.jpg") 11 11 11 11 stretch;
 border-image: url("images/border.jpg") 11 11 11 11 stretch;
}
p.two {
 border-style: solid;
 border-width: 20px;
 -moz-border-image: url("images/border.jpg") 11 11 11 11 round;
 -webkit-border-image: url("images/border.jpg") 11 11 11 11 round;
 border-image: url("images/border.jpg") 11 11 11 11 round;
}

CSS3: 方框陰影

box-shadow屬性。
語法: box-shadow: h-shadow v-shadow blur spread color inset;。
前兩個值不能省略，所以box-shadow至少為前兩個值加上一個顏色值。
h-shadow，就是horizontal offset，負數值會將陰影放在方框左側。
v-shadow，就是vertical offset，負數值會將陰影放在方框上方。
blur，就是blur distance，如果略過不寫，陰影就會呈現像邊框一樣的實線。
spread，就是spread of shadow，正數值會使陰影往所有方向伸展，負數值則會將陰影縮減。
inset，將外陰影轉為內陰影。
-moz-box-shadow和-webkit-box-shadow並不在CSS規格內，但可以幫助舊版的Chrome、Firefox和safari。

body {
 text-align: center;
}
p.one {
 border-style: solid;
 border-width: 5px;
 -moz-border-image: -5px -5px #777777;
 -webkit-border-image: -5px -5px #777777;
 border-image: -5px -5px #777777;
}
p.two {
 border-style: solid;
 border-width: 5px;
 -moz-border-image: 5px 5px 5px #777777;
 -webkit-border-image: 5px 5px 5px #777777;
 border-image: 5px 5px 5px #777777;
}
p.three {
 border-style: solid;
 border-width: 5px;
 -moz-border-image: 5px 5px 5px 5px #777777;
 -webkit-border-image: 5px 5px 5px 5px #777777;
 border-image: 5px 5px 5px 5px #777777;
}
p.four {
 border-style: solid;
 border-width: 5px;
 -moz-border-image: 0 0 10px #777777;
 -webkit-border-image: 0 0 10px #777777;
 border-image: 0 0 10px #777777;
}
p.five {
 border-style: solid;
 border-width: 5px;
 -moz-border-image: 0 0 10px #777777 inset;
 -webkit-border-image: 0 0 10px #777777 inset;
 border-image: 0 0 10px #777777 inset;
}

CSS3: 圓角

border-radius屬性。用像素來指定半徑尺寸。
語法: border-radius: 5px, 10px, px, 10px;(順時鐘方向:上、右、下、左)。
可以使用下列屬性為方框的各邊角指定個別的值: border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius、border-top-left-radius。 -moz-box-shadow和-webkit-box-shadow並不在CSS規格內，但可以幫助舊版的Chrome、Firefox和safari。

body {
 text-align: center;
}
p.one {
 border-style: solid;
 border-color: #cccccc;
 border-width: 5px;
 -moz-border-radius: 10px;
 -webkit-border-radius: 10px;
 border-radius: 10px;
}

CSS3: 橢圓形

語法: border-radius: 80px 50px;(水平和垂直的距離)。
可以使用下列屬性為方框的各邊角指定個別的值: border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius、border-top-left-radius。
也可以使用速記法，一次指派四個水平和四個垂直值: border-radius: 1em 4em 1em 4em / 2em 1em 2em 1em;。
你可以在一個正方形的方框中，將border-radius設為和方框相同的高度，就可以做出圓形。

body {
 text-align: center;
}
p {
 border-style: solid;
 border-color: #cccccc;
 border-width: 5px;
} p.one {
 border-top-left-radius: 80px 50px;
 -moz-border-radius-top-left: 80px 50px;
 -webkit-border-radius-top-left: 80px 50px;
} p.one {
 border-radius: 1em 4em 1em 4em / 2em 1em 2em 1em;
 -moz-border-radius: 1em 4em 1em 4em / 2em 1em 2em 1em;
 -webkit-border-radius: 1em 4em 1em 4em / 2em 1em 2em 1em;
}
