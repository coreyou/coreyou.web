在CSS中控制影像尺寸

指定影像尺寸可以幫助網頁下載更順利，因為HTML和CSS通常會比影像還早下載，
所以告訴瀏覽器要預留多少空間給影像，就可以讓它先處理剩餘頁面而不需要等待影像下載完畢。
很多網站使用的影像都是相同的，因為透過格線設計，會有一些常用的尺寸，例如:
小型垂直：220x360
小型水平：330x210
焦點照片：620x400
要在網站中使用相同尺寸的影像，可以用CSS來控制，而非將尺寸放在HTML裡面。

img.large {
 width: 500px;
 height: 500px;
}

使用CSS對齊影像_靠左靠右

使用float屬性對齊影像。

img.align_left {
 float: left;
 margin-right: 10px;
}
img.align_right {
 float: right;
 margin-right: 10px;
}

使用CSS對齊影像_置中

影像是行內元素，這代表它們會隨著周遭的文字流動。為了將影像置中，就必須使用值為block的display屬性，將它轉成區塊層級元素。
轉換成區塊層級元素以後，有以下兩種常用的方式可以將影像水平置中：

  1. 在容器元素中，可以使用text-align: center;
  2. 在影像本身上，可以使用margin，將左右邊界都設為auto
指定影像尺寸和對齊方式的技巧也可以使用在<figure>上。

img.align_center {
 display: block;
 margin: 0px auto;
}

背景影像

background-image屬性可以在任何HTML元素背後加入影像，可以是整個頁面，也可以是局部頁面。
背景影像通常是頁面中最後載入的(這會使得網站看起來下載緩慢)。

body {
 background-image: url("images/pattern.gif");
}
p {
 background-image: url("images/pattern.gif");
}

重複影像

background-repeat: repeat(背景影像會以水平方向和垂直方向重複，如果沒使用background-repeat屬性，則為預設值)、repeat-x(水平方向重複)、repeat-y(垂直方向重複)、no-repeat(影像只顯示一次)。
background-attachment: fixed(背景影像會固定在同一個位置)、scroll(背景影像會隨著使用者上下捲動頁面而移動)。

body {
 background-image: url("images/pattern.gif");
 background-repeat: repeat-x;
}
body {
 background-image: url("images/pattern.gif");
 background-repeat: no-repeat;
 background-attachment: fixed;
}

背景位置

background-position屬性：當影像不重複的時候，可以用這個屬性來指定背景影像應該放在瀏覽器視窗何處，屬性值有兩個，第一個是水平位置，第二個是垂直位置。
水平位置值可以是left或center或right，垂直位置值可以是top、center、bottom。
如果只指定一個值，第二個值就會預設為center。
也可以使用像素或是百分比來指定水平與垂直位置，代表與瀏覽器視窗左上角的距離。比如background-position: 50% 50%;，就是指水平垂直都置中。

body {
 background-image: url("images/pattern.gif");
 background-repeat: no-repeat;
 background-position: center top;
}

/*百分比位置*/
body {
 background-image: url("images/pattern.gif");
 background-repeat: no-repeat;
 background-position: 50% 50%;
}

背景速記

background屬性：依照以下順序指定值

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position
CSS3也可以透過重複background速記的方式，來支援多重背景影像，但不是所有瀏覽器都支援。

body {
 background: #ffffff url("images/pattern.gif") no-repeat top right;
}

/*多重背景*/
body {
 background: url("images/pattern.gif") top right no-repeat, url("images/pattern.gif") top left no-repeat;
}

變換影像和SPRITE

製作一個當使用者滑鼠移上時會換到第二個樣式的連結或按鈕，作法是將一個影像設為背景，這個影像包含連結或按鈕的三個不同狀態。
使用SPRITE的優點是瀏覽器只需要要求單張影像而非許多影像，這可以讓網頁下載速度變快。

a.button {
 height: 44px;
 background-image: url("images/img_navsprites_hover.gif");
 display: inline-block;
}
a#home {
 left: 0px;
 width: 46px;
 background-position: 0px 0px;
}
a#prev {
 left: 63px;
 width: 43px;
 background-position: -47px 0px;
}
a#next {
 left: 129px;
 width: 43px;
 background-position: -91px 0px;
}
a#home:hover {
 background-position: 0 -45px;
}
a#prev:hover {
 background-position: -47px -45px;
}
a#next:hover {
 background-position: -91px -45px;
}
a#home:active {
 background-position: 0px 0px;
}
a#prev:active {
 background-position: -47px 0px;
}
a#next:active {
 background-position: -91px 0px;
}

CSS3: 漸層

body {
 /* fallback color */
 background-color: #66cccc;
 /* fallback image */
 background-image: url("images/ground control.jpg");
 /* Firefox 3.6+ */
 background-image: -moz-linear-gradient(#336666, #66cccc);
 /* Safari 4+, Chrome 1+ */
 background-image: -webkit-gradient(linear, 0% 0%, 0% 100%, from(#66cccc), to(#336666));
 /* Safari 5.1+, Chrome 10+ */
 background-image: -webkit-linear-gradient(#336666, #66cccc);
 /* Opera 11.10+ */
 background-image: -o-linear-gradient(#336666, #66cccc);
 background-repeat: no-repeat;
}
