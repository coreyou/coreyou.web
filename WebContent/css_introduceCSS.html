選取器和宣告、屬性和值

語法: CSS的語法如下

h1, h2, h3 {
 font-family: Arial;
 color: yellow;
}

以上語法規定了<h1>、<h2>、<h3>的樣式。h1, h2, h3是選取器(selector)；大括號裡面是宣告；宣告中，冒號以前是屬性，冒號以後是值。

使用外部CSS

語法: 在<head>內，使用<link>連結到CSS檔案。

using_external_css.html HTML
<!DOCTYPE html>
<html>
 <head>
  <title>Using External CSS</title>
  <link href="css/using_external_css.css" type="text/css" rel="stylesheet" />
 </head>
 <body>
  <h1>Potatoes</h1>
  <p>There are dozens of different potato varieties. They are usually described as early. second early and maincrop.</p>
 </body>
</html>

using_external_css.css CSS
body {
 font-family: arial;
 background-color: rgb(185,179,175);
}
h1 {
 color: rgb(255,255,255);
}

使用內部CSS

語法:

using_external_css.html HTML
<!DOCTYPE html>
<html>
 <head>
  <title>Using Internal CSS</title>
  <style type="text/css">
   body {
    font-family: arial;
    background-color: rgb(185,179,175);
   }
   h1 {
    color: rgb(255,255,255);
   }
  </style>
 </head>
 <body>
  <h1>Potatoes</h1>
  <p>There are dozens of different potato varieties. They are usually described as early. second early and maincrop.</p>
 </body>
</html>

CSS選取器

語法: 選擇器有大小寫區別

選取器 意義 範例
全域選取器 套用到文件的所有元素上 * {}
找出頁面上所有元素
類型選取器 找出符合的元素名稱 h1, h2, h3 {}
找出<h1>、<h2>和<h3>元素
CLASS選取器 找出class屬性值符合句號後所指定的值之元素 .note {}
找出任何class屬性值為note的元素
p.note {}
找出class屬性值為note的<p>元素
ID選取器 找出id屬性值符合#符號後所指定的值之元素 #introduction {}
找出id屬性值為introduction的元素
子元素選取器 找出一元素之直系子元素 li>a {}
找出任何身為<li>元素之子的<a>元素
後代選取器 找出一指定元素之後代元素(不僅是直系子元素) p a {}
找出任何位在<p>元素內的<a>元素，即使兩者之間還有堆疊了其他元素
相連手足選取器 找出一元素之相連手足元素 h1+p {}
找出<h1>元素之後的第一個<p>元素(但排除其他<p>元素)
整體手足選取器 找出一元素之手足元素，即使它並非直接相連的元素 h1~p {}
如果<h1>元素有兩個<p>元素，此規則會套用到兩者上

CSS如何規範階層

規則:

 
1.後來先贏
 
如果兩個選取器相同的話，後者優先。
2.指定程度
 
較具指定性的規則優先。例如:
h1比*更具指定性、p b比p更具指定性、p#intro比p更具指定性。

繼承

如果你指定<body>元素的font-family或color屬性，也會套用到大部分的子元素上。
然而像是background-color或border屬性並不會繼承給子元素。
你可以在屬性中使用inherit值，來強迫許多屬性從父元素那裡繼承屬性值。例如:

body {
 font-family: Arial, Verdana, sans-serif;
 color: #665544;
 padding: 10px;}
.page {
 border: 1px solid #665544;
 background-color: #efefef;
 padding: inherit;}

class="page" 的元素的內距尺寸會套用到<body>元素的CSS規則。
