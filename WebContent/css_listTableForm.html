項目符號樣式

list-style-type屬性：
可以讓你控制項目符號的形狀或樣式。
可以用在套用到<ol>、<ul>、<li>元素的規則上。
在項目清單上可以使用下列屬性值: none、disc、circle、square。
在編號清單上可以使用下列屬性值: decimal、decimal-leading-zero、lower-alpha、upper-alpha、lower-roman、upper-roman。

ul.two {
 list-style-type: circle;
}

使用圖片作為項目符號

list-style-image屬性。
語法: list-style-image: url("images/list_icon.png");
可以用在套用到<ul>、<li>元素的規則上。
下例也示範了margin屬性的用法，增加了清單中的每個項目的垂直間距。

ul {
 list-style-image: url("images/list_icon.png");
}
li {
 margin: 10px 0px 0px 0px;
}

項目符號的位置

list-style-position屬性。指定了項目符號應該出現在包含項目之方框的裡面還是外面。
語法: list-style-position: outside(符號位在文字區塊的左側，是預設值)、inside(符號位在文字方框內部，會縮排);

ul.one {
 list-style-position: outside;
}
ul.two {
 list-style-position: inside;
}

空白儲存格的邊框

empty-cells屬性，指定空白儲存格的邊框是否要顯示。
empty-cells: show(顯示空白儲存格的邊框)、hide(隱藏空白儲存格的邊框)、inherit(當一個表格位在另一個表格內的時候，內部的表格會繼承外部表格容器的規則)。

table.one {
 empty-cells: show;
} table.two {
 empty-cells: hide;
}

儲存格之間的間距

border-spacing屬性，控制相連儲存格之間的距離。
border-spacing: 5px 15px;(水平和垂直間距，也可以統一只寫出一個間距值)。
當儲存格使用邊框的時候，在兩個儲存格相交處，線條應該是正常外緣的兩倍粗，可以使用border-collapse屬性來解決此情況。
border-collapse: collapse(邊框交疊成單一邊框，border-spacing、empty-cell都會被忽略)、separate(邊框會互相分開，會遵守border-spacing、empty-cell)。

table.one {
 border-spacing: 5px 15px;
} table.two {
 border-collapse: collapse;
}

對齊表單控制項: 問題與解決方案

問題:
 表單元素的標籤通常長度不一，這代表表單控制項不會對齊成一直線。
 在範例中，像使用者詢問的每個主題都被個別放在一個<div>元素中，以確保每個問題會出現在新的一行。如果表單控制項對齊一條垂直線的話，使用者會比較方便填寫。
 如果你看一下我們詢問性別的那一欄，兩個單選鈕都有自己的<label>。這裡在標題上加了<span>元素來協助對齊這些控制項。
解決方案:
 在文字輸入欄中，標題位在<label>元素裡；在單選鈕中，標題是在<span>元素上。兩者都有一個class屬性，值為title。
 我們可以使用名為float的屬性來將標題移到頁面左側。
 設定這些元素的width屬性，我們就可以確定這些標題會占用同樣的寬度，這樣一來，它們旁邊的表單控制項就會對齊。
 這裡用了text-align屬性將標題靠右對齊，以及padding屬性來確認標題方框裡的文字和表格控制項之間會有空格。
 容納了表單各行的<div>元素也套用了樣式(固定其寬度並製作每列之間的垂直間格)。最後送出按鈕也設定靠右對齊。

<form action="example.php" method="post">
 <div>
  <label for="name" class="title">Name:</label>
  <input type="text" id="name" name="name" />
 </div>
 <div>
  <label for="email" class="title">Email:</label>
  <input type="email" id="email" name="email" />
 </div>
 <div>
  <span class="title">Gender:</span>
  <input type="radio" name="gender" id="male" value="M" />
  <label for="male">Male</label>
  <input type="radio" name="gender" id="female" value="F" />
  <label for="female">Female</label><br />
 </div>
 <div>
  <input type="submit" value="Register" id="submit" />
 </div>
