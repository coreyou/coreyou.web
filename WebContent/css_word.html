指定字型

font-family屬性，可以指定一連串字型，之間用逗號分開，如果使用者未安裝你首選的字型，瀏覽器會嘗試從清單中找到一個替代字型。
如果字型名稱裡有兩個字，那就應該將它放在雙引號中。

可以寫在font-family屬性內的字體專有名詞:
serif(櫬線): Georgia, Times, Times New Roman
sans-serif(無襯線): Arial, Verdana, Helvetica
monospace(等寬): Courier, Courier New
曲線字: Comic Sans MS, Monotype Corsiva
狂想: Impact, Haettenschweiler

body {
 font-family: Georgia, Times, serif;
}
h1, h2 {
 font-family: Arial, Verdana, sans-serif;
}
.credits {
 font-family: "Courier New", Courier, monospace;
}

字體大小

font-size屬性，可以使用以下三種方式來指定字體大小:
1. 像素，例如:12px。
2. 百分比，瀏覽器預設文字大小是16px，所以75%等於12px，200%等於32px。如果你將<body>元素裡的所有文字都設成75%(12px)，然後又指派<body>元素裡的某個元素內容設為75%，則該元素會變成9px。
3. EM，一個em等於m字母的寬度。

像素大約等於pt，因為1pt等於1/72吋，而大部分的電腦解析度為72ppi。

更多字型選擇與字型格式

@font-face可以連結到某個字型檔的路徑，設定一個使用者電腦中沒有的字型，如果使用者的電腦沒有該字型，就會進行下載。
其中需要以ont-family設定其字型名、src設定字型檔路徑、format設定字型格式。
不同的瀏覽器支援不同的字型格式，字型格式應該以EOT、WOFF、TTF/OTF、SVG的順序出現在原始碼中。
字型轉檔的網站 Font Squirrel也會提供@font-face的CSS code，可以直接複製到程式碼內。

@font-face {
 font-family: ‘pacificoregular’
 src: url(‘fonts/pacifico_web/pacifico-webfont.eot’);
 src: url(‘fonts/pacifico_web/pacifico-webfont.eot?#iefix’) format(‘embedded-opentype’),
  url(‘fonts/pacifico_web/pacifico-webfont.woff’) format(‘woff’),
  url(‘fonts/pacifico_web/pacifico-webfont.ttf’) format(‘truetype’),
  url(‘fonts/pacifico_web/pacifico-webfont.svg#pacificoregular’) format(‘svg’);
 font-weight: normal;
 font-style: normal;
}
h1, h2 {
 font-family: pacificoregular;
}

CSS3: 陰影

text-shadow: 1px 1px 0px #000000;
屬性的值為三個長度、一個顏色。第一個長度為左右位移值；第二個長度為上下位移值；第三個為陰影的模糊量，是選擇性的；第四個為陰影的顏色。

p.two {
 background-color: #eeeeee;
 color: #666666;
 text-shadow: 1px 1px 0px #000000;
}

連結的樣式化

CSS中有兩個擬類別可以設定已造訪和未造訪連結的不同樣式。
:link 可以設定未造訪連結的樣式。
:visited 可以設定已被點按之連結的樣式。

a:link {
 color: deeppink;
 text-decoration: none;
}
a:visited {
 color: black;
}

對使用者的回應

CSS中有三個擬類別可以在使用者與元素互動時，改變這些元素的外觀。
:hover 使用者將滑鼠移到連結或按鈕等元素上面時。
:visited 當按鈕或連結等元素被點按時。
:focus 當游標位於表單欄位等元素時。

在使用擬元素時，應該以下列順序出現：:link、:visited、:hover、:focus、:active。

input {
 padding: 6px 12px 6px 12px;
 border: 1px solid #665544;
 color: #ffffff;
}
 input.submit:hover {
 background-color: #665544;
}
input.submit:active {
 background-color: chocolate;
}
input.text {
 color: #cccccc;
}
input.text:focus {
 color: #665544;
}

屬性選取器

選取器 意義 範例
EXISTENCE []
找出一個特定屬性(不管值為何) 		p[class]
找出任何帶有class屬性的<p>元素
EQUALITY [=]
找出一個有特定值的特定屬性 		p[class="dog"]
找出class屬性值為dog的任何<p>元素
SPACE [~=]
找出一個特定屬性，其值出現在一串用空格分開的字詞當中 		p[class~="dog"]
找出class屬性值是一串空白隔開的字詞清單，而且其中一個字是dog的任何<p>元素
PREFIX [^=]
找出一個特定屬性，其值的開頭為某一特定字串 		p[attr^"d"]
找出class屬性值是以字母d為開頭的任何<p>元素
SUBSTRING [*=]
找出一個特定屬性，其值包含了一個特定子字串 		p[attr*"do"]
找出class屬性值中包含了do這個字詞的任何<p>元素
SUFFIX [$=]
找出一個特定屬性，其值的結尾為某一特定字串 		p[attr$"g"]
找出class屬性值是以字母g為結尾的任何<p>元素
