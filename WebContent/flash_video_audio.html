將FLASH影片加到網頁上

從Google免費取得SWFObject的程式碼，使用了<div>元素來製作一個Flash影片的空間，id屬性被SWFObject程式碼使用，在head有兩個<script>元素，
第一個<script>中，type屬性表示裡面的程式碼是以JavaScript寫成的，src屬性告訴瀏覽器到哪裡尋找這份程式碼；
第二個<script>是用來告訴瀏覽器關於此Flash的資訊，embedSWF()中的五個參數，分別是:
1.swf檔的位置。
2.Flash影片應該取代的元素，透過<div>元素的id屬性來辨識。
3.Flash影片的寬度。
4.Flash影片的長度。
5.觀看此影片需要的最低版本Flash播放器: Flash Play 8。

語法:

<!DOCTYPE html>
<html>
 <head>
  <meta charset="UTF-8">
  <title>Flash, Video, Audio</title>
  <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/swfobject/2.2/swfobject.js"></script>
  <script type="text/javascript">swfobject.embedSWF("flash/over.swf", "over", "400", "300", "8.0.0");</script>
 </head>
 <body>
  <div id="over"><p>看不到影片的人，會看到這行字，這行字也可以讓視障人士讀取。</p></div>
 </body>
</html>

將FLASH視訊加到網頁上

三步驟: 1.將視訊轉為FLV格式。2.找到可以撥放視訊的FLV播放器(www.osflv.com提供免費開放原始碼播放器)。3.將播放器和視訊加到頁面中。
第一個<script>中，type屬性表示裡面的程式碼是以JavaScript寫成的，src屬性告訴瀏覽器到哪裡尋找這份程式碼；
第二個<script>是用來告訴瀏覽器關於此Flash的資訊，flashvar變數用不到所以留白，param提供影片路徑，embedSWF告訴程式要用視訊播放器取代HTML元素。

語法:

<!DOCTYPE html>
<html>
 <head>
  <meta charset="UTF-8">
  <title>Flash, Video, Audio</title>
  <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/swfobject/2.2/swfobject.js"></script>
  <script type="text/javascript">
   var flashvars = {};
   var params = {movie:"../ground control.flv"};
   swfobject.embedSWF("flash/OSFlvPlayer_v4.2/OSplayer.swf", "groundControl", "400", "320", "8.0.0", flashvars, params);
  </script>
 </head>
 <body>
  <div id="groundControl"><p>KIMULA的ground Control LIVE</p></div>
 </body>
</html>

HTML5:將視訊加到網站

瀏覽器支援的格式:
H264:IE和Safari。
WebM:Android、Chrome、Firefox、Opera。
播放器的控制鈕是由瀏覽器控制，你也可以使用JavaScript來改變控制鈕的外觀。
使用Flash Player播放器的問題是，當有選單之類的元素覆蓋其上，或者視窗放大縮小時，它的行為就不容易預測，HTML5可以解決這些問題。

語法: 將以下放在body內。

<video src="視訊路徑" poster="開始播放前或視訊正在下載時所顯示的影像"
 width="400" height="300"
 preload="有三種值:none(在使用者案播放鍵之前，瀏覽器不應該載入視訊)、auto(瀏覽器應該在頁面載入的同時也下載視訊)、metadata(瀏覽器只需要收集大小、第一個影格、播放清單和片長之資訊)">
 controls(使用了此屬性會指定瀏覽器提供其本身的播放控制鈕)
 autoplay(使用了此屬性會指定檔案應該自動播放)
 loop(使用了此屬性會指定視訊應該在結束之後從頭播放一次)
</video>

controls、autoplay、loop等屬性就像是開關，存在就會開啟功能，不一定需要提供一個值。

HTML5:數個視訊來源

語法: 在<video>中使用<source>，來取代<video>開始標記中的src屬性，可以使用多個<source>來標明該視訊有各種不同格式可以使用。

<video poster="images//ground control.jpg" width="400" height="320" preload controls >
 <source src="video/ground control.mp4" type=‘video/mp4;codecs="avc1.42E01E, mp4a.40.2"’ />
 <source src="video/ground control.webM" type=‘video/webm;codecs="vp8, vorbis"’ />
 <p>如果瀏覽器不支援，就會顯示這段內容</p>
</video>

加入FLASH MP3播放器

語法:

<!DOCTYPE html>
<html>
 <head>
  <meta charset="UTF-8">
  <title>Flash, Video, Audio</title>
  <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/swfobject/2.2/swfobject.js"></script>
  <script type="text/javascript">
   var flashvars = {};
   var params = {mp3:"../audio/Sleep Away.mp3"};
   swfobject.embedSWF("flash/player_mp3_maxi.swf", "demoMusic", "200", "20", "8.0.0", flashvars, params);
  </script>
 </head>
 <body>
  <div id="demoMusic"><p>You cannot hear this track because this browser does not support our Flash music player.</p></div>
 </body>
</html>

You cannot hear this track because this browser does not support our Flash music player.

語法: <audio>

<audio src="音訊檔的路徑" controls(播放器是否應該顯示控制按鈕，可以自己用JavaScript來指定自己的控制按鈕) autoplay(自動播放) preload(和video的preload一樣有三種值) loop(是否重複播放) >
